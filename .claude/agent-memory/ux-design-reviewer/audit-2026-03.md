## Critical Patterns Found
1. Typography inconsistency: Schools/Favorites/Profile/Onboarding use Geist Sans `font-black` h1s instead of Oswald `text-display`
2. FavoriteButton touch target: h-8 w-8 (32px) below iOS 44pt minimum everywhere
3. drop-cap class misused on short program descriptions (overview-tab, campus-tab)
4. Academics tab: stat VALUES colored in school primary (bad for dark schools)
5. No iOS safe area insets on any fixed/sticky elements
6. Profile and program-manager pages break design language entirely (generic shadcn)
7. Jersey builder has no theatrical header or entrance animation
8. Register button turns blue when Coach role selected (not in palette)
9. Platform main: flat p-6 with no responsive variation
10. Mobile hamburger floats independently, not anchored to topbar
